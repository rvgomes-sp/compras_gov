14.6. Consultar Unidade

Serviço que permite consultar uma unidade pertencente a um órgão/entidade a partir de seu código.

14.6.1. Detalhes da Requisição

Endpoint

Método HTTP

/v1/orgaos/{cnpj}/unidades/{codigoUnidade}

GET

14.6.2. Exemplo Requisição (cURL)

1      curl -k -X GET \
2        "${BASE_URL}/v1/orgaos/10000000000003/unidades/1" \
3        -H "accept: */*"
