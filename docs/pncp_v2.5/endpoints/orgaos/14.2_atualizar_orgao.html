14.2. Atualizar Órgão

Serviço destinado exclusivamente aos administradores do PNCP que permite atualizar os dados de um órgão/entidade no repositório do PNCP que eventualmente esteja desatualizado.

Atenção

Não está disponível para plataformas privadas a partir de 18/08/2025.

14.2.1. Detalhes da Requisição

Endpoint

Método HTTP

/v1/orgaos

PUT

14.2.2. Exemplo de Payload

1{
2   "cnpj": "10000000000003"
3}

14.2.3. Exemplo Requisição (cURL)

1curl -k -X PUT \
2  --header "Authorization: Bearer access_token" \
3  "${BASE_URL}/v1/orgaos" \
4  -H "accept: */*" \
5  -H "Content-Type: application/json" \
6  --data "@/home/objeto.json"
