11.12. Consultar Contratos de Ata de Registro de Preço

Serviço que permite consultar todos os contratos/empenhos vinculados a uma ata de registro de preço específica.

11.12.1. Detalhes da Requisição

Endpoint

Método HTTP

/v1/orgaos/{cnpj}/compras/{ano}/{sequencial}/atas/{sequencialAta}/contratos

GET

11.12.2. Exemplo de Payload

1      Não se aplica

11.12.3. Exemplo Requisição (cURL)

1     curl -k -X GET \
2       "${BASE_URL}/v1/orgaos/10000000000003/compras/2021/1/atas/1/contratos" \
3       -H "Accept: */*"

11.12.4. Dados de entrada

Nota
