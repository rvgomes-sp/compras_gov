11.16. Consultar Partes Envolvidas em Ata de Registro de Preço

Serviço que recupera a lista de partes envolvida em uma ata de registro de preço específica.

11.16.1. Detalhes da Requisição

Endpoint

Método HTTP

/v1/orgaos/{cnpj}/compras/{anoCompra}/{sequencialCompra}/atas/{sequencialAta}/partesenvolvidas

GET

11.16.2. Exemplo de Payload

1      Não se aplica

11.16.3. Exemplo Requisição (cURL)

1     curl -X GET \
2       "${BASE_URL}/v1/orgaos/00394460000141/compras/2021/1/atas/1/partesenvolvidas" \
3       -H "Accept: */*"

11.16.4. Dados de entrada

Nota
