# Documentação PNCP 2.5

Fontes locais usadas para construir e auditar o GSB Monitor V2.5.

- `manual/`: Manual de Integração PNCP 2.5 completo.
- `openapi/`: especificação OpenAPI integral.
- `AUDITORIA_PACOTES_ENDPOINTS.md`: integridade, cobertura e decisões de arquivamento.

O Manual completo e o OpenAPI integral são as únicas fontes documentais PNCP versionadas.
Recortes HTML por número de seção foram removidos porque duplicavam essas fontes e
permitiam regressão para uma visão parcial da API.

Endpoints de escrita permanecem apenas como documentação no OpenAPI e não são
autorizados no monitor.
