# Documentação PNCP 2.5

Fontes locais usadas para construir e auditar o GSB Monitor V2.5.

- `manual/`: Manual de Integração PNCP 2.5 completo.
- `openapi/`: especificação OpenAPI integral.
- `endpoints/compras/`: consultas de contratação, itens, resultados, documentos e histórico.
- `endpoints/atas/`: consultas de atas, histórico, contratos, partes e documentos.
- `endpoints/orgaos/`: consultas e referências de órgãos e unidades.
- `AUDITORIA_PACOTES_ENDPOINTS.md`: integridade, cobertura e decisões de arquivamento.

Os recortes individuais facilitam consulta, mas o Manual completo é a referência documental integral. Endpoints de escrita permanecem apenas como documentação e não são autorizados no monitor.
