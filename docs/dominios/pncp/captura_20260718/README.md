# Fotografia das tabelas de domínio PNCP

Captura datada de 18/07/2026, preservada para auditoria e comparação. O arquivo `_indice.json` registra origem, quantidade, tamanho e SHA-256 de cada tabela.

As tabelas com origem `API` e `MANUAL` são fotografias, não contratos imutáveis. Antes de orientar uma decisão comercial ou regulatória, a versão vigente deve ser confirmada na fonte oficial.
