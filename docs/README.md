# Fontes oficiais canônicas

Cada fonte externa possui um único documento técnico integral. Recortes de
páginas, cópias por endpoint e arquivos auxiliares não são versionados.

| Fonte | Documento canônico | Papel |
|---|---|---|
| PNCP | `pncp_v2.5/manual/` e `pncp_v2.5/openapi/` | Contratação, item, resultado e evidência do evento |
| CNPJ | `cnpj/openapi/api-docs.json` | Enriquecimento cadastral do fornecedor |
| Obrasgov | `obrasgov/openapi/api-docs.json` | Vínculo e acompanhamento de projetos de infraestrutura |
| Compras.gov | `catalogo_compras_gov/` | Catálogo oficial de serviços |
| Domínios PNCP | `dominios/pncp/` | Fotografias datadas das tabelas de domínio |

A seleção operacional de endpoints pertence a `config/api/`. A documentação
integral prova o contrato da fonte, mas não autoriza automaticamente coleta,
escrita ou qualificação comercial.
