# API Obrasgov

Fonte documental oficial para vincular oportunidades a projetos de investimento
em infraestrutura e acompanhar sua execução.

- OpenAPI: `openapi/api-docs.json`
- título: `Dados Abertos - Obrasgov.br`
- versão: `1.0.15`
- servidor documentado: `https://api.obrasgov.gestao.gov.br/obrasgov/api`
- situação operacional: documentada; vínculo automático ainda não ativado

As sete consultas cobrem projeto de investimento, geometria, execução física,
arquivos da intervenção, execução financeira, saldo contábil e contrato.
Ausência de vínculo no Obrasgov não invalida uma oportunidade do PNCP.
