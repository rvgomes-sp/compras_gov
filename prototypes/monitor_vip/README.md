# Monitor

Protótipo da interface do monitor comercial, preservado sem feeds, amostras, resultados ou bases do V2.

O arquivo `index.html` e seus ativos só poderão consumir uma saída nova, registrada e aprovada pelo GSB Monitor V2.5. Nenhum `monitor_feed_real.json` ou `monitor_feed_mock.json` foi versionado. Esta pasta não integra o fluxo operacional.
